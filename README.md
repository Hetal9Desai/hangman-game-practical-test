Hangman Coding Quiz Game

A simple interactive Hangman-style quiz game built with HTML, CSS, and TypeScript/JavaScript. This game challenges users to guess programming-related words using hints and a visual hangman indicator, all within a limited time.

## Features

    Guess programming-related terms with helpful hints.

    Visual hangman that updates with each wrong guess.

    Countdown timer (30 seconds per round).

    Keyboard and on-screen button input support.

    Responsive UI with end-of-game modals for win/loss.

## Getting Started

1. Clone the Repository

git clone https://github.com/Hetal9Desai/hangman-game-practical-test
cd hangman-game-practical-test

2. Open index.html

You can simply open the index.html file in your web browser:

open index.html

Go To Live Server

🛠️ Technologies Used

    HTML – Structure of the page

    CSS – Styling and layout

    TypeScript / JavaScript – Game logic and interactivity

## How to Play

    A word is chosen randomly from a predefined quiz list.

    Use your keyboard or click the on-screen buttons to guess letters.

    Each incorrect guess updates the hangman image.

    You have 6 wrong guesses and 30 seconds to guess the full word.

    Game ends with a success or failure modal.
